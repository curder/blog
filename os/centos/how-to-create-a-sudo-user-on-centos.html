# 如何在CentOS上创建sudo用户

# 介绍

该sudo命令提供了一种机制，用于向普通用户授予通常仅对root用户可用的管理员权限。

下面将展示在CentOS上创建具有sudo访问权限的新用户的最简单方法，而无需修改服务器的sudoers文件。

# 以root用户身份登录服务器

ssh root@server_ip_address
1

# 使用adduser命令添加新用户到系统

这里以用户username，密码以newPassword为例。

echo "newPassword" | passwd --stdin "username"
1

** 强烈建议使用强密码！**

# 将用户添加到wheel组

默认情况下，在CentOS上，该wheel组的成员具有sudo权限，使用usermod命令：

usermod -aG wheel username
1

# 在新用户上测试sudo访问权限

  • 使用su命令切换到新用户帐户
su - username
1
  • 列出目录的内容，/root通常只有root用户可以访问。
sudo ls -la /root
1

第一次sudo在会话中使用时，系统将提示您输入用户帐户的密码。输入密码以继续。
