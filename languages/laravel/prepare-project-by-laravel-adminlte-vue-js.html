# 使用Laravel + adminlte + vue.js构建项目

英文原文地址：@lojorider/prepare-project-by-laravel-adminlte-vue-js-644cab072909open in new window

# 本地开发环境

  • Laravel 5.6.16
  • PHP 7.1.13
    • extension=php_openssl.dll
    • extension=php_pdo_mysql.dll
  • MySQL 5.7.30
  • Laravel Packages
    • acacha/adminlte-laravelopen in new window
    • laravel/duskopen in new window
    • laravelcollective/htmlopen in new window
    • Maatwebsite/Laravel-Excelopen in new window
    • intervention/imageopen in new window
    • tymondesigns/jwt-authopen in new window

# 创建项目

composer create-project --prefer-dist laravel/laravel adminlte-laravel "5.6.*"
1

执行完上面的命令将生成一个以adminlte-laravel为目录的laravel项目

# 设置admin-lte包

在项目adminlte-laravel根目录下执行命令

composer require "acacha/admin-lte-template-laravel" \
&& php artisan vendor:publish --tag=adminlte --force \
&& composer require "laravel/dusk" --dev \
&& php artisan make:adminUserSeeder \
&& composer require "laravelcollective/html" \
&& composer dump-autoload

# 修改数据库配置

数据库配置文件为：config\database.php

'charset' => env('DB_CHARSET', 'utf8mb4'),
'collation' => env('DB_COLLATION', 'utf8mb4_unicode_ci'),

2

# 配置本地.env

mysql数据库的配置不支持utf8mb4

DB_CHARSET=utf8
DB_COLLATION=utf8_unicode_ci
1
2

同时修改其他数据库配置信息，如:

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=homestead
DB_USERNAME=homestead
DB_PASSWORD=secret
