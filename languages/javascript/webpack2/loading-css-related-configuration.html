# 加载CSS的相关配置

CSS相关的loader包含css-loader和style-loader。

其中css-loader主要是处理@import和url()，如果@import加载的是外部资源，那么就跳过，只处理本地资源。 而style-loader处理style标签。

# 安装

npm i css-loader style-loader --save-dev
1

# 配置

const config = {
  // ...
  module: {
    rules: [
      // ...
      {
        test: /\.css$/,
        exclude: /node_modules/,
        use: [
          'style-loader',
          {
            loader: 'css-loader',
            options: {
              modules: true,
            },
          },
        ],
      ],
    },
  },
};

// ...
import './main.css';
