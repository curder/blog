# webpack2介绍与安装

WebPack可以看做是模块打包机：它做的事情是，分析项目结构，找到JavaScript模块以及其它的一些浏览器不能直接运行的拓展语言（Sass，TypeScript等），并将其转换和打包为合适的格式供浏览器使用。

# webpack2的安装

安装webpack2之前首先要安装NodeJsopen in new window，在MacOSX上推荐使用nvm管理NodeJS和NPM，测试安装结果可以使用：

node -v # 查看NodeJS版本
v8.11.2

npm -v # 查看npm版本
5.6.0

# 初始化一个项目

在使用webpack之前，使用npm的命令初始化一个项目。

npm init -y
1

生成package.json文件，内容如下：

{
  "name": "webpack2-study",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": [],
  "author": "",
  "license": "ISC"
}

有了这个文件，可以使用下面的命令安装webpack2.7.0。

npm install webpack@2 --save-dev
1

参数 –save-dev 是要保存到package.json中在开发时使用这个包，而生产环境中不使用。

# 生成压缩文件

新建一个文件app/index.js，内容如下：

console.log("hello world");
1

使用 webpack 命令打包

./node_modules/.bin/webpack app/index.js dist/app.js
1
